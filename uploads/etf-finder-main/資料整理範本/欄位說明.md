# ETF 資料整理範本 — 欄位說明

## 如何使用

1. 用 Google Sheet 開啟 → **檔案 → 匯入 → 上傳** `etf_template.csv`
2. 匯入時選「**取代目前的工作表**」、分隔符選「**逗號**」
3. 開始一行一行填 ETF 資料（一支 ETF = 一行）
4. 整理完後，**檔案 → 下載 → 逗號分隔值 (.csv)**
5. 把 CSV 用 `轉換腳本.py` 轉成 `etf.json`（腳本見同資料夾）

---

## 欄位定義

| 欄位 | 必填 | 格式 | 範例 | 說明 |
|---|---|---|---|---|
| `code` | ✅ | 字串 | `0050`、`00891`、`00679B` | ETF 代碼，**前導 0 要保留**（Google Sheet 要把這欄設成「純文字」格式，否則 0050 會被吃成 50） |
| `name` | ✅ | 字串 | `元大台灣50` | ETF 中文全名 |
| `asset_class` | ✅ | 列舉值 | `股票` | 見下方「資產大類列舉」 |
| `asset_mid` | ✅ | 字串 | `區域` / `產業` / `風格` / `種類` / `期限` | 中分類 |
| `asset_subclass` | ✅ | 字串，多項用 ` \| ` 分隔 | `股票-區域-新興亞太-台灣 \| 股票-產業-科技` | 一支 ETF 可同時屬於多個細類，用 **空格+直線+空格** 分隔 |
| `investment_type` | ✅ | 列舉值 | `市值型` | 見下方「投資分類列舉」 |
| `active_passive` | ✅ | 列舉值 | `被動` | `主動` 或 `被動` |
| `leverage` | ✅ | 列舉值 | `一般` | `一般` / `2x槓桿` / `-1x反向` / `-2x反向` |
| `theme_factor` | ⬜ | 字串，多項用 `, ` 分隔 | `ESG, 半導體` | 主題/因子標籤；沒有就**留空** |
| `dividend_months` | ⬜ | 數字，多項用 `, ` 分隔 | `1, 4, 7, 10` | 配息月份；無配息**留空** |
| `top5_holdings` | ✅ | 字串，5 個用 `, ` 分隔 | `台積電, 鴻海, 聯發科, 廣達, 中信金` | 前 5 大持股 |
| `investment_direction` | ✅ | 字串 | 追蹤台灣50指數… | 100 字以內的投資方向說明 |
| `official_url` | ✅ | URL | `https://...` | 官方資訊連結 |

### 中分類（asset_mid）
依大類不同：
- **股票** → `區域` / `產業` / `風格`
- **債券** → `種類` / `區域` / `期限`
- **原物料** → `貴金屬` / `能源` / `農產品` / `工業金屬`
- **外匯** → `幣別`
- **槓桿反向** → `槓桿` / `反向`

## Google Sheet 設定建議

1. **第一欄（code）設定為「純文字」格式**
   - 選 A 欄 → 格式 → 數字 → **純文字**
   - 否則 `0050` 會被當成數字顯示成 `50`

2. **凍結首列**
   - 檢視 → 凍結 → 1 列

3. **設定資料驗證**（避免打錯字）
   - 選 `asset_class` 欄 → 資料 → 資料驗證 → 條件「下拉選單」
   - 輸入：股票, 債券, 原物料, 外匯, 槓桿反向
   - 其他列舉值欄位同理

4. **建議色彩標示**
   - 必填欄位（標題列）填粉紅色
   - 選填欄位填灰色

## 資料來源

整理 ETF 資料時可參考：

| 用途 | 資料來源 |
|---|---|
| ETF 清單、基本資料 | 投信投顧公會 SITCA → 「ETF 專區」 |
| 上市/上櫃分類 | TWSE / TPEX 官網 |
| 持股、配息月份 | 各發行商網站（元大、國泰、富邦、群益、中信、復華、群益…） |
| 投資方向說明 | 各 ETF 公開說明書摘要 |

## 常見錯誤

❌ `code = 50`（被數字化）→ ✅ `code = "0050"`（純文字）
❌ `asset_subclass = 股票-區域-新興亞太-台灣，股票-產業-科技`（用中文逗號）→ ✅ 用 ` | ` 分隔
❌ `dividend_months = 1月, 4月, 7月, 10月` → ✅ 純數字 `1, 4, 7, 10`
❌ `theme_factor = null` → ✅ 直接留空
